# README — Informe Técnico General del Sistema

## 1. Propósito del documento

Este repositorio/carpeta contiene la estructura base para construir el **informe técnico general del sistema**.  
El documento tiene como objetivo consolidar los aportes de diferentes participantes sobre los componentes técnicos del proyecto.

Este informe **no corresponde directamente al trabajo de grado final**, sino que servirá como documento base para organizar, documentar e integrar la información técnica del sistema.

Cada participante debe aportar una sección específica siguiendo una estructura común, con el fin de facilitar posteriormente el ensamblaje del informe final.

---

## 2. Alcance del informe

El informe general documenta el sistema desde una perspectiva técnica, integrando:

- Requerimientos y restricciones.
- Diseño general del sistema.
- Hardware.
- Procesamiento en borde.
- Plataforma web en la nube.
- Análisis espectral y localización.
- Protocolos de prueba.
- Técnica de trabajo colaborativo usando agentes IA.

El documento busca explicar **qué se hizo, cómo se diseñó, cómo funciona, cómo se prueba y qué limitaciones tiene el sistema**.

---
