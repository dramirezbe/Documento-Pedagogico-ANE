\chapter{Referencias}

Esta sección debe reunir las fuentes técnicas, académicas y documentales utilizadas
para sustentar decisiones de diseño, selección de tecnologías, métodos de análisis
y procedimientos de implementación o prueba.

Como placeholder, este bloque ayuda a reservar la estructura necesaria para incorporar
referencias bibliográficas consistentes y trazables, siguiendo el formato de citación
que el equipo o la institución definan posteriormente.

\subsection*{Puntos sugeridos para completar}
\begin{itemize}[leftmargin=*]
  \item Artículos, libros o reportes técnicos relevantes.
  \item Documentación oficial de hardware, SDR y plataformas en nube.
  \item Referencias sobre algoritmos de análisis espectral y localización.
  \item Normas, manuales o estándares aplicables.
  \item Formato de citación adoptado por el documento final.
\end{itemize}
