\chapter{Referencias}

Esta secci\'on debe reunir las fuentes t\'ecnicas, acad\'emicas y documentales utilizadas
para sustentar decisiones de dise\~no, selecci\'on de tecnolog\'ias, m\'etodos de an\'alisis
y procedimientos de implementaci\'on o prueba.

Como placeholder, este bloque ayuda a reservar la estructura necesaria para incorporar
referencias bibliogr\'aficas consistentes y trazables, siguiendo el formato de citaci\'on
que el equipo o la instituci\'on definan posteriormente.

\subsection*{Puntos sugeridos para completar}
\begin{itemize}[leftmargin=*]
  \item Art\'iculos, libros o reportes t\'ecnicos relevantes.
  \item Documentaci\'on oficial de hardware, SDR y plataformas en nube.
  \item Referencias sobre algoritmos de an\'alisis espectral y localizaci\'on.
  \item Normas, manuales o est\'andares aplicables.
  \item Formato de citaci\'on adoptado por el documento final.
\end{itemize}
