\section{Hardware}

Esta secci\'on est\'a destinada a documentar los elementos f\'isicos del sistema: tarjetas,
sensores, antenas, fuentes de alimentaci\'on, interconexiones y cualquier componente
relevante para el montaje o funcionamiento de la plataforma.

El placeholder permite reservar la estructura necesaria para luego incorporar fichas
t\'ecnicas, criterios de selecci\'on de componentes y observaciones derivadas de pruebas
o integraci\'on del hardware.

\subsection*{Puntos sugeridos para completar}
\begin{itemize}[leftmargin=*]
  \item Inventario de componentes de hardware.
  \item Criterios de selecci\'on y compatibilidad.
  \item Diagrama de conexi\'on o montaje.
  \item Consumo energ\'etico, robustez y limitaciones f\'isicas.
  \item Riesgos t\'ecnicos asociados al hardware.
\end{itemize}
