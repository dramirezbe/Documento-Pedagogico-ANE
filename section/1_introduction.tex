\chapter{Introducci\'on general}

Esta secci\'on debe presentar el contexto del proyecto, el problema que se busca resolver
y la motivaci\'on del sistema. Tambi\'en sirve para explicar, de forma breve, cu\'al es el
alcance del documento y c\'omo se relacionan sus partes principales.

Como placeholder, este texto puede reemplazarse posteriormente por una narrativa m\'as
formal que describa el objetivo del sistema, los actores involucrados y el valor t\'ecnico
de la soluci\'on propuesta.

\subsection*{Puntos sugeridos para completar}
\begin{itemize}[leftmargin=*]
  \item Descripci\'on del problema o necesidad principal.
  \item Objetivo general del sistema.
  \item Alcance del documento t\'ecnico.
  \item Resumen corto de la arquitectura o m\'odulos principales.
  \item Relaci\'on entre borde, nube y an\'alisis posterior.
\end{itemize}
