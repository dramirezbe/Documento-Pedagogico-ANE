\section{SDR: adquisici\'on y preprocesamiento de se\~nales de radio en borde}

Esta secci\'on debe explicar c\'omo se capturan las se\~nales usando SDR, qu\'e etapas de
preprocesamiento se ejecutan en el borde y cu\'ales son las salidas intermedias que luego
se env\'ian o publican para an\'alisis posterior.

Como placeholder, el texto actual define una estructura inicial para documentar pipelines
de captura, filtrado, segmentaci\'on, almacenamiento temporal y optimizaciones para trabajo
en tiempo real o cuasi-tiempo real.

\subsection*{Puntos sugeridos para completar}
\begin{itemize}[leftmargin=*]
  \item Configuraci\'on del SDR y par\'ametros de adquisici\'on.
  \item Flujo de preprocesamiento en borde.
  \item Estrategias de limpieza, filtrado o reducci\'on de ruido.
  \item Formato de salida de datos y m\'etodos de transmisi\'on.
  \item Limitaciones operativas y oportunidades de mejora.
\end{itemize}
